About

This web application uses machine learning models trained on a dataset of 28,619 news headlines from TheOnion (sarcastic) and HuffPost (non-sarcastic). Three different RNN architectures are available for comparison:

  • SimpleRNN: Basic recurrent neural network (fastest, baseline performance)
  • LSTM: Long Short-Term Memory network (better at long-term dependencies)
  • BiLSTM: Bidirectional LSTM (best performance, processes text in both directions)

All processing happens in your browser. Your data never leaves your device.

Dataset: News Headlines Dataset for Sarcasm Detection | Built with TensorFlow.js
